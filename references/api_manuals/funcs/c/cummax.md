# cummax

## 语法

`cummax(X)`

参数说明和窗口计算规则请参考：[累计窗口系列（cum 系列）](../themes/cumFunctions.md)

## 详情

计算 *X* 元素的累计最大值。

## 例子

```
x = [7,4,5,8,9]
cummax(x);
// output
[7,7,7,8,9]

m = matrix(6 5 7 8 1, 3 9 4 2 10);
m;
```

| #0 | #1 |
| --- | --- |
| 6 | 3 |
| 5 | 9 |
| 7 | 4 |
| 8 | 2 |
| 1 | 10 |

```
cummax(m);
```

| #0 | #1 |
| --- | --- |
| 6 | 3 |
| 6 | 9 |
| 7 | 9 |
| 8 | 9 |
| 8 | 10 |
