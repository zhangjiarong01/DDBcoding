# imax

## 语法

`imax(X)`

## 参数

**X** 可以是标量、向量、矩阵或表。

## 详情

* 若 *X* 是向量，返回最大值的位置。如果有多个相同的最大值，返回左起第一个最大值的位置。
* 若 *X* 为矩阵，计算在每列内部进行，返回一个向量。
* 若 *X* 为表，计算在每列内部进行，返回一个表。

## 例子

```
x = 1.2 2 NULL 6 -1 6;
imax(x);
// output
3

x = 5 3 1 6 4 6 $ 3:2;
imax(x);
// output
(0,1)

x=array(INT,0);
x;
// output
[]

imax(x);
// output
-1
// 对于一个空向量，imax 返回 -1

m=matrix(1 2 3, 6 5 4);
m;
```

| #0 | #1 |
| --- | --- |
| 1 | 6 |
| 2 | 5 |
| 3 | 4 |

```
imax(m);
// output
[2,0]
```
