# tmavg

## 语法

`tmavg(T, X, window)`

参数说明和窗口计算规则请参考：[tmFunctions](../themes/tmFunctions.md)

## 详情

在给定长度（以时间 *T* 衡量）的滑动窗口内计算 *X* 元素的平均值。

## 例子

```
T = 1 1 1 2 5 6
X = 1 4 2 -1 2 4
m = table(T as t,X as x)
select *, tmavg(t, x, 3) from m
```

| t | x | tmavg\_t |
| --- | --- | --- |
| 1 | 1 | 1 |
| 1 | 4 | 2.5 |
| 1 | 2 | 2.3333 |
| 2 | -1 | 1.5 |
| 5 | 2 | 2 |
| 6 | 4 | 3 |

```
T = 2021.01.02 2021.01.02  2021.01.04  2021.01.05 2021.01.07 2021.01.08
X = 1 4 2 -1 2 4
m = table(T as t,X as x)
select *, tmavg(t, x, 3d) from m
```

| t | x | tmavg\_t |
| --- | --- | --- |
| 2021.01.02 | 1 | 1 |
| 2021.01.02 | 4 | 2.5 |
| 2021.01.04 | 2 | 2.3333 |
| 2021.01.05 | -1 | 0.5 |
| 2021.01.07 | 2 | 0.5 |
| 2021.01.08 | 4 | 3 |

```
select *, tmavg(t, x, 1w) from m
```

| t | x | tmavg\_t |
| --- | --- | --- |
| 2021.01.02 | 1 | 1 |
| 2021.01.02 | 4 | 2.5 |
| 2021.01.04 | 2 | 2.3333 |
| 2021.01.05 | -1 | 1.5 |
| 2021.01.07 | 2 | 1.6 |
| 2021.01.08 | 4 | 2 |
