# cumstdp

## 语法

`cumstdp(X)`

参数说明和窗口计算规则请参考：[累计窗口系列（cum 系列）](../themes/cumFunctions.md)

## 详情

计算 *X* 元素的累计总体标准差。

## 例子

```
x = [1,2,4,NULL,8];
cumstdp(x)
// output
[0, 0.5, 1.247219128924647, 1.247219128924647, 2.680951323690902]
```

```
m=matrix(0.15 0.08 0.03 -0.14 -0.09, 0.2 -0.12 -0.16 0.08 0.16);
m;
```

| 0.08 | -0.12 |

```
cumstdp(m);
```
